# 🧘 Zen Lab - Public Releases

Welcome to the public download repository for **Zen Lab**! 

Zen Lab is a premium, beautifully crafted personal sanctuary mobile app designed to help you cultivate mindfulness, establish daily rituals, build habits, and visualize your personal growth in a digital garden.

---

## 📥 Download the App

The source code of the project is kept private, but you can download the latest compiled installation packages (APKs) directly from the releases page:

👉 **[Go to the Latest Releases Page](https://github.com/Predatormk11/zen-lab-releases/releases)**

1. Click on the latest release tag (e.g. `v1.0.0`).
2. Under the **Assets** section, click on `zen-lab.apk` to download the file directly to your Android device.

---

## 📱 Features

* **Sanctuary Dashboard**: A calming space to check your daily mindfulness goals and track your progress.
* **Rituals & Habits**: Easily create customizable daily habits and rituals with modern visual widgets.
* **Digital Garden**: Watch your growth flourish in an interactive visual garden representation of your habits.
* **Zen Wrapped**: Beautiful analytical summaries highlighting your dedication, streak milestones, and achievements.
* **Privacy First**: Zen Lab works completely offline-first using a fast, secure local SQLite database, keeping your data entirely yours.

## ⚙️ Installation Instructions (Android)

Because this APK is distributed directly and not downloaded through the Google Play Store, you will need to allow your device to install external applications:

1. **Download the APK** from the link above.
2. Tap on the downloaded file.
3. If prompted with a warning about installing apps from **"Unknown Sources"** (or your browser/file manager):
   * Go to **Settings** on your phone.
   * Toggle **"Allow from this source"** to enable installation.
4. Go back to the APK installation prompt and select **Install**.
5. Open Zen Lab and start your journey!
